# Contacts

Projeto criado para exemplificar passos de como tornar sua aplicaçāo mais facil de se testar.

Neste projeto foram usadas as seguintes ferramentas:

* ViewModel
* Coroutines
* DataBinding
* Koin
* LiveData
* Robolectric
* Mockk
